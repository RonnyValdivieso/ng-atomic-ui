Input Component

Componente de entrada de texto reutilizable y accesible con soporte para diferentes tipos, estados, validaciones y eventos.

Características Principales

Tipos de Input

text, email, password, number, tel, url

Estados

normal, disabled, loading, error, success

Tamaños

small, medium (default), large

Validación

required, custom validation, error messages

Tipos de Input

Texto type="text"
Email type="email"
Contraseña type="password"
Número type="number"
Teléfono type="tel"
URL type="url"

Tamaños

Pequeño size="small"
Mediano (Por defecto) size="medium"
Grande size="large"

Estados

Normal estado: normal
Disabled disabled="true"
Loading loading="true"
Error error="true"
Success success="true"
Required required="true"

Validación y Mensajes de Error

Email con Validación

Por favor, ingresa un email válido

Contraseña con Validación

La contraseña debe tener al menos 8 caracteres

Campo Exitoso

Nombre válido

Casos de Uso Comunes

Formulario de Registro

Nombre completo
Correo electrónico
Contraseña

Búsqueda

Teléfono

Ejemplo de Implementación

<!-- Input básico -->
<app-input 
  type="text"
  placeholder="Ingresa tu texto aquí...">
</app-input>

<!-- Input con validación -->
<app-input 
  type="email"
  placeholder="correo@ejemplo.com"
  required="true"
  error="true">
</app-input>

<!-- Input con estados -->
<app-input 
  type="password"
  placeholder="Contraseña"
  size="large"
  loading="true">
</app-input>

<!-- Con label asociado -->
<app-label for="username">Nombre de usuario</app-label>
<app-input 
  id="username"
  type="text"
  placeholder="usuario123"
  required="true">
</app-input>
