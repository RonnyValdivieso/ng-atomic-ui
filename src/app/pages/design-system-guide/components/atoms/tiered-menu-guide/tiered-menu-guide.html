TieredMenu Component

Componente de menú desplegable multinivel para navegación jerárquica y organización de acciones. Soporta menús anidados, iconos, separadores y múltiples configuraciones.

Características Principales

Multinivel

Soporte para menús anidados

3 Tamaños

small, medium, large

2 Variantes

default, outlined

Modos

Popup y estático

Casos de Uso

Menús de Aplicación

Ideal para menús principales como File, Edit, View típicos de aplicaciones de escritorio.

Menús de Usuario

Perfecto para opciones de perfil, configuración y preferencias del usuario.

Navegación Jerárquica

Excelente para estructuras de navegación con múltiples niveles de categorías.

Menús Contextuales

Útil para acciones específicas del contexto con opciones organizadas por categorías.

Mejores Prácticas

Hacer

  • ✓ Agrupa acciones relacionadas lógicamente
  • ✓ Usa separadores para dividir grupos distintos
  • ✓ Incluye iconos para facilitar el reconocimiento
  • ✓ Limita la profundidad a 3 niveles máximo
  • ✓ Usa labels descriptivos y concisos
  • ✓ Implementa callbacks para acciones

Evitar

  • ✗ Menús con más de 4 niveles de anidamiento
  • ✗ Demasiados elementos en un solo nivel
  • ✗ Labels ambiguos o muy largos
  • ✗ Mezclar acciones sin organización lógica
  • ✗ Omitir separadores entre grupos diferentes
  • ✗ Usar para listas largas de datos (usar dropdown)

Ejemplos de Código

TypeScript - Definición de Items

import { MenuItem } from 'primeng/api';

items: MenuItem[] = [
  {
    label: 'Archivo',
    icon: 'pi pi-file',
    items: [
      {
        label: 'Nuevo',
        icon: 'pi pi-plus',
        items: [
          { label: 'Documento', icon: 'pi pi-file' },
          { label: 'Hoja de cálculo', icon: 'pi pi-table' }
        ]
      },
      { label: 'Abrir', icon: 'pi pi-folder-open' },
      { separator: true },
      { label: 'Guardar', icon: 'pi pi-save', command: () => this.save() }
    ]
  },
  {
    label: 'Editar',
    icon: 'pi pi-pencil',
    items: [
      { label: 'Copiar', icon: 'pi pi-copy' },
      { label: 'Pegar', icon: 'pi pi-paste' }
    ]
  }
];

HTML - Implementación

<!-- Modo Popup -->
<app-button (clicked)="menu.toggle($event)" icon="pi pi-bars">
  Menú
</app-button>
<app-tiered-menu 
  #menu
  [model]="items"
  [popup]="true"
  size="medium"
  variant="default"
  (itemClick)="handleMenuClick($event)">
</app-tiered-menu>

<!-- Modo Estático -->
<app-tiered-menu 
  [model]="items"
  [popup]="false"
  variant="outlined">
</app-tiered-menu>

TypeScript - Manejo de Eventos

handleMenuClick(event: { originalEvent: Event; item: MenuItem }) {
  console.log('Item clicked:', event.item.label);
  
  // Ejecutar acción específica
  if (event.item.command) {
    event.item.command(event);
  }
}

Referencia de API

Prop Tipo Default Descripción
model MenuItem[] [] Array de elementos del menú
popup boolean true Si el menú es popup o estático
size 'small' | 'medium' | 'large' 'medium' Tamaño del menú
variant 'default' | 'outlined' 'default' Variante visual
disabled boolean false Deshabilita el menú
appendTo any 'body' Elemento al que se adjunta el popup
autoZIndex boolean true Gestión automática del z-index
baseZIndex number 0 Z-index base para el menú

Eventos

Evento Tipo Descripción
itemClick { originalEvent: Event; item: MenuItem } Se emite al hacer click en un elemento
menuShow void Se emite cuando el menú se muestra
menuHide void Se emite cuando el menú se oculta

Métodos Públicos

Método Parámetros Descripción
toggle(event) event: Event Alterna la visibilidad del menú (solo popup)
show(event) event: Event Muestra el menú (solo popup)
hide() - Oculta el menú (solo popup)
