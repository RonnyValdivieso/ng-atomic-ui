Checkbox Component

Componente fundamental para selección múltiple. Incluye soporte para estado indeterminado, múltiples tamaños, variantes y colores con accesibilidad completa.

Características Principales

2 Variantes

default, outlined

3 Tamaños

small, medium, large

Estados

checked, unchecked, indeterminate, disabled

3 Colores

primary, success, danger

Tamaños Disponibles

size="small"
size="medium"
size="large"

Variantes Disponibles

variant="default"
variant="outlined"

Colores Disponibles

color="primary"
color="success"
color="danger"

Estados del Checkbox

[checked]="false"
[checked]="true"
[indeterminate]="true"
[disabled]="true"

Ejemplo Interactivo

Checkbox Básico

Estado: {{ basicChecked() ? 'Aceptado' : 'No aceptado' }}

Checkbox Required

Estado: {{ termsAccepted() ? 'Aceptado ✓' : 'Requerido *' }}

Estado Indeterminado (Select All)

Estado del grupo: @if (indeterminateState()) { Parcialmente seleccionado } @else if (selectAll()) { Todo seleccionado } @else { Nada seleccionado }

Mejores Prácticas

Recomendado

  • • Usar labels descriptivos y claros
  • • Agrupar checkboxes relacionados
  • • Usar estado indeterminado para "select all"
  • • Indicar claramente campos requeridos con *
  • • Mantener consistencia en tamaños y colores
  • • Proveer feedback visual al interactuar

Evitar

  • • Checkboxes sin label visible
  • • Labels ambiguos o poco descriptivos
  • • Demasiados checkboxes sin organizar
  • • Usar checkbox para selección única (usar radio)
  • • Mezclar tamaños sin razón visual clara
  • • Checkboxes disabled sin explicación

Ejemplos de Código

<!-- Checkbox básico -->
<app-checkbox 
  label="Acepto los términos"
  [checked]="accepted"
  (checkedChange)="onAccept($event)">
</app-checkbox>

<!-- Checkbox con variante y color -->
<app-checkbox 
  label="Suscribirse al newsletter"
  variant="outlined"
  color="success"
  [checked]="subscribed"
  (checkedChange)="onSubscribe($event)">
</app-checkbox>

<!-- Checkbox requerido -->
<app-checkbox 
  label="He leído la política de privacidad"
  [required]="true"
  [checked]="privacyAccepted"
  (checkedChange)="onPrivacyChange($event)">
</app-checkbox>

<!-- Checkbox indeterminado (Select All) -->
<app-checkbox 
  label="Seleccionar todo"
  [checked]="allSelected"
  [indeterminate]="someSelected"
  (checkedChange)="onSelectAll($event)">
</app-checkbox>
