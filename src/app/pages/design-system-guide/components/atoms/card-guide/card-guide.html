Tamaños Disponibles

Este es un card pequeño con padding reducido, ideal para espacios limitados o información concisa. size="small"
Este es un card de tamaño mediano, el tamaño por defecto y más común para la mayoría de los casos de uso. size="medium"
Este es un card grande con padding generoso, perfecto para contenido importante o destacado. size="large"

Variantes Disponibles

Card con sombra sutil y borde. Esta es la variante por defecto. variant="default"
Card sin sombra con borde más pronunciado. Más discreto visualmente. variant="outlined"
Card sin borde con sombra pronunciada. Destaca del fondo. variant="elevated"

Mejores Prácticas

Recomendado

  • • Usar cards para agrupar información relacionada
  • • Mantener consistencia en elevaciones en toda la app
  • • Usar hoverable para cards clickeables
  • • Incluir header descriptivo y claro
  • • Usar tamaños apropiados según el contexto
  • • Aplicar spacing consistente entre cards

Evitar

  • • Anidar cards dentro de cards repetidamente
  • • Usar elevaciones muy altas sin motivo
  • • Sobrecargar cards con demasiada información
  • • Mezclar variantes sin criterio visual
  • • Hacer cards demasiado anchos sin contenido
  • • Olvidar espaciado entre elementos internos

Ejemplos de Código

<!-- Card básico -->
<app-card header="Título del Card">
  Contenido del card aquí
</app-card>

<!-- Card completo -->
<app-card 
  header="Artículo"
  subheader="Publicado hoy"
  footer="Por Juan • 5 min"
  size="medium"
  variant="elevated"
  [elevation]="2"
  [hoverable]="true">
  Contenido del artículo...
</app-card>

<!-- Card outlined -->
<app-card 
  header="Información"
  variant="outlined">
  Contenido con borde
</app-card>

<!-- Card sin padding para imágenes -->
<app-card 
  header="Galería"
  [noPadding]="true">
  <img src="imagen.jpg" class="w-full" />
</app-card>

Referencia de API

Propiedad Tipo Default Descripción
header string '' Título del card
subheader string '' Subtítulo o descripción
footer string '' Contenido del footer
size 'small' | 'medium' | 'large' 'medium' Tamaño del card
variant 'default' | 'outlined' | 'elevated' 'default' Variante visual
elevation 0 | 1 | 2 | 3 | 4 | 5 1 Nivel de sombra
hoverable boolean false Activa efecto hover
noPadding boolean false Elimina padding interno
