Badge

Indicadores versátiles para notificaciones, estados, contadores y etiquetas con múltiples variantes y opciones de posicionamiento

Características Principales

Múltiples Variantes

Cuatro estilos disponibles: solid, outline, soft y dot para diferentes niveles de énfasis visual

Contadores Inteligentes

Formateo automático de contadores con valores máximos personalizables y opción de mostrar cero

Posicionamiento Flexible

Sistema de posicionamiento en cuatro esquinas para badges de notificación superpuestos

Animación y Interactividad

Soporte para animación de pulso y manejadores de eventos click con estados disabled
