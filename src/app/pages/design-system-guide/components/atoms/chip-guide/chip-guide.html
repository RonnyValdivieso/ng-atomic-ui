Chip Component

Componente compacto para mostrar etiquetas, contactos o filtros.

Características Principales

Etiquetas

Visualización compacta de datos.

Media

Soporte para iconos e imágenes.

Removible

Opción para eliminar el chip.

Estilos

Adaptable al tema y modo oscuro.

Ejemplos de Uso

Básico

Con Iconos

Con Imágenes

Removible

Ejemplos de Código

<!-- Básico -->
<app-chip label="Action"></app-chip>

<!-- Con Icono -->
<app-chip label="Facebook" icon="pi pi-facebook"></app-chip>

<!-- Removible -->
<app-chip 
	label="Tag 1" 
	[removable]="true" 
	(onRemove)="onRemove($event)">
</app-chip>
