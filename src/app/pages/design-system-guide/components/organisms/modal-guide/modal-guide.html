Modal Component

Componente para mostrar contenido en una capa superior, bloqueando la interacción con el fondo. Ideal para diálogos de confirmación, formularios y mensajes importantes.

Características Principales

Flexible

Soporta contenido arbitrario y footers personalizados.

Posicionamiento

Top, Bottom, Left, Right, Center.

Responsive

Se adapta a diferentes tamaños de pantalla.

Theming

Soporte nativo para modo oscuro.

Uso Básico

Un modal simple con título y contenido.

Ejemplos de Código

<!-- Básico -->
<app-modal [(visible)]="isVisible" header="Título">
  <p>Contenido del modal...</p>
</app-modal>

<!-- Con Footer -->
<app-modal [(visible)]="isVisible" header="Confirmar">
  <p>¿Estás seguro?</p>
  <ng-template footer>
    <p-button label="Cancelar" (click)="isVisible.set(false)"></p-button>
    <p-button label="Aceptar" (click)="confirm()"></p-button>
  </ng-template>
</app-modal>
