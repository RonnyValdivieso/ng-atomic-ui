Form Field

Molécula que combina Label e Input para crear campos de formulario completos con validación, accesibilidad e integración con formularios reactivos

Características Principales

Diseño Atómico

Composición siguiendo principios de Atomic Design usando átomos Label + Input

Validación Integrada

Sistema de validación completo con mensajes de error y estados visuales automáticos

Reactive Forms

Integración completa con Angular Reactive Forms usando ControlValueAccessor

Accesibilidad Completa

ARIA attributes, asociación label-input automática y soporte completo de teclado

Tamaños

Tres variantes de tamaño para adaptarse a diferentes contextos y jerarquías visuales.

Small Uso compacto
Medium Por defecto
Large Alto énfasis

Tipos de Input

Múltiples tipos de input semánticos para diferentes tipos de datos y validación automática.
