Breadcrumb Component

Componente de navegación que indica la ubicación actual del usuario dentro de la jerarquía de la aplicación.

Características Principales

Jerarquía

Visualización clara de la ruta de navegación.

Home

Icono de inicio configurable.

Enlaces

Integración con RouterLink y eventos de clic.

Iconos

Soporte para iconos en cada elemento.

Con Iconos

Ejemplos de Código

<!-- Básico -->
<app-breadcrumb
  [items]="items"
  [home]="home">
</app-breadcrumb>

<!-- TypeScript -->
items: MenuItem[] = [
  {{ '{' }} label: 'Dashboard' {{ '}' }},
  {{ '{' }} label: 'Settings' {{ '}' }}
];
home: MenuItem = {{ '{' }} icon: 'pi pi-home', routerLink: '/' {{ '}' }};
