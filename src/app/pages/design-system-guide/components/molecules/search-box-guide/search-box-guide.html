Search Box

Molécula sofisticada que combina Input y Button para crear una interfaz de búsqueda con debounce, estados de carga y funcionalidad de limpieza.

Características Principales

Composición Atómica

Combina átomos Input, Button e Icon siguiendo principios de Atomic Design.

Debounce

Sistema de debounce configurable para reducir llamadas API y mejorar rendimiento.

Estados de Carga

Indicadores visuales de carga para operaciones asíncronas de búsqueda.

Limpieza Rápida

Botón de limpieza integrado con soporte completo de teclado (Escape).

Mejores Prácticas

✅ Hacer

  • • Usar búsqueda con debounce para reducir llamadas API
  • • Proporcionar estados de carga claros para operaciones asíncronas
  • • Incluir placeholders descriptivos
  • • Manejar estados de búsqueda vacía con gracia
  • • Usar tamaños apropiados para diferentes contextos

❌ No Hacer

  • • Hacer la búsqueda demasiado sensible
  • • Deshabilitar funcionalidad de limpieza sin razón
  • • Usar tiempos de debounce muy cortos (<200ms)
  • • Olvidar manejar navegación por teclado
  • • Ignorar estados de carga para búsquedas lentas
