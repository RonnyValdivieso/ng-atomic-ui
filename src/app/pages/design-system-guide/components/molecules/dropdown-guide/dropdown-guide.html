Dropdown Component

Componente de selección que permite elegir una opción de una lista. Soporta filtrado, limpieza y validación de formularios.

Características Principales

Selección

Selección simple de opciones.

Filtrado

Búsqueda integrada para listas largas.

Formularios

Integración nativa con Reactive Forms.

Theming

Diseño consistente y modo oscuro.

Ejemplos de Uso

Básico

Selected: {{ selectedCity() || 'None' }}

Reactive Forms

Estados

Ejemplos de Código

<!-- Básico con ngModel -->
<app-dropdown
  [options]="cities"
  optionLabel="name"
  optionValue="code"
  [(ngModel)]="selectedCity"
  label="City">
</app-dropdown>

<!-- Reactive Forms con Error -->
<app-dropdown
  [options]="cities"
  optionLabel="name"
  [formControl]="cityControl"
  label="City"
  [error]="cityControl.invalid ? 'Required' : ''">
</app-dropdown>
