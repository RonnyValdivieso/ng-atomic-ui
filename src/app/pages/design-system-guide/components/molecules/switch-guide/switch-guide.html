Switch Component

Componente de alternancia para activar o desactivar una opción. Ideal para configuraciones y preferencias.

Características Principales

Toggle

Alternancia simple de estados booleanos.

Etiquetas

Soporte integrado para etiquetas clicables.

Formularios

Integración nativa con Reactive Forms.

Valores

Soporte para valores personalizados (true/false).

Ejemplos de Uso

Básico

Status: {{ checked() ? 'Enabled' : 'Disabled' }}

Reactive Forms

Valores Personalizados

Current Value: {{ status() }}

Estado Deshabilitado

Ejemplos de Código

<!-- Básico con ngModel -->
<app-switch
  [(ngModel)]="checked"
  label="Enable Notifications">
</app-switch>

<!-- Reactive Forms con Error -->
<app-switch
  [formControl]="termsControl"
  label="Accept Terms"
  [error]="termsControl.invalid ? 'Required' : ''">
</app-switch>
