Toast Component

Notificaciones no intrusivas que proporcionan feedback inmediato sobre las operaciones del sistema.

Características Principales

Feedback Visual

Colores semánticos para éxito, error, advertencia e información.

Temporizado

Cierre automático configurable o modo "sticky".

Posicionamiento

Ubicación flexible en cualquier esquina o centro.

Theming

Diseño consistente con soporte para modo oscuro.

Ejemplos de Uso

Haz clic en los botones para probar los diferentes tipos de notificaciones.

Ejemplos de Código

// Inyectar MessageService
constructor(private messageService: MessageService) {{ '{' }}

// Mostrar notificación
this.messageService.add({{ '{' }} 
  severity: 'success', 
  summary: 'Éxito', 
  detail: 'Operación completada' 
{{ '}' }});

// Notificación persistente
this.messageService.add({{ '{' }} 
  severity: 'info', 
  summary: 'Aviso', 
  detail: 'Mensaje importante',
  sticky: true 
{{ '}' }});
