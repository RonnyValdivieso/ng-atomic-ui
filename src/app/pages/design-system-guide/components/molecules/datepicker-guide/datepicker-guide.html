Datepicker Component

Componente para la selección de fechas y horas. Soporta rangos, formatos personalizados y validación.

Características Principales

Selección

Selección simple, múltiple y por rangos.

Hora

Soporte opcional para selección de hora.

Formularios

Integración nativa con Reactive Forms.

Configurable

Formatos, iconos y restricciones de fecha.

Ejemplos de Uso

Básico

Selected: {{ date() | date:'mediumDate' }}

Rango de Fechas

Start: {{ rangeDates()?.[0] | date:'shortDate' }} - End: {{ rangeDates()?.[1] | date:'shortDate' }}

Reactive Forms

Restricciones (Min/Max)

Ejemplos de Código

<!-- Básico con ngModel -->
<app-datepicker
  [(ngModel)]="date"
  label="Select Date">
</app-datepicker>

<!-- Rango de Fechas -->
<app-datepicker
  [(ngModel)]="rangeDates"
  selectionMode="range"
  label="Select Range">
</app-datepicker>
