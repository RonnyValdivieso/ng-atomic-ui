Guía de Implementación

Ejemplos de código y mejores prácticas para implementar el sistema de temas en tu aplicación.

Uso Recomendado en CSS 

/* ✅ Forma recomendada - Usa variables PrimeNG */
.mi-componente {{ '{' }}
  background: var(--p-surface-0);
  color: var(--p-text-color);
  border: 1px solid var(--p-surface-300);
  transition: all 0.3s ease;
{{ '}' }}

/* ❌ Evita valores hardcoded */
.mi-componente {{ '{' }}
  background: #ffffff; /* No se adapta */
  color: #000000;      /* No se adapta */
{{ '}' }}

Integración con Angular 

// Servicio de tema
@Injectable({{ '{' }} providedIn: 'root' {{ '}' }})
export class ThemeService {{ '{' }}
  private _isDarkMode = signal<boolean>(false);
  readonly isDarkMode = this._isDarkMode.asReadonly();
  
  toggleDarkMode(): void {{ '{' }}
    this._isDarkMode.update(current => !current);
    document.documentElement.classList
      .toggle('dark-theme');
  {{ '}' }}
{{ '}' }}
